# Changelog

All notable changes to `webshr/bricks-stubs` are documented here. Tags mirror
the upstream Bricks Builder version the stubs were generated from.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html)
mirrored against the upstream Bricks release.

## [Unreleased]

## [2.3.3] - 2026-04-27

### Added

- Initial public release.
- Stubs generated from Bricks Builder 2.3.3 via
  [php-stubs/generator](https://github.com/php-stubs/generator).
- `composer generate` script and `stubs-finder.php` for regeneration.
- PHPStan usage example in README.
